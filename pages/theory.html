Theory Reference

Language architecture — C, Python & Java at a glance.

Hello, World

C and Java need an explicit entry point and compile step; Python runs a script top to bottom.

C
#include <stdio.h> int main(void) { printf("Hello, World!\n"); return 0; }
PYTHON
print("Hello, World!")
JAVA
public class Main { public static void main(String[] args) { System.out.println("Hello, World!"); } }

Variables & Types

C and Java are statically typed (checked at compile time); Python is dynamically typed (checked at run time).

C
int age = 21; double price = 9.99; char grade = 'A';
PYTHON
age = 21 price = 9.99 grade = "A" # no type keyword needed
JAVA
int age = 21; double price = 9.99; char grade = 'A';

Loops

The classic for-loop shape is nearly identical in C and Java. Python favors iterating a range or collection directly.

C
for (int i = 0; i < 5; i++) { printf("%d\n", i); }
PYTHON
for i in range(5): print(i)
JAVA
for (int i = 0; i < 5; i++) { System.out.println(i); }

Functions

C and Java require a declared return type (or void). Python functions are defined with def and return dynamically.

C
int square(int n) { return n * n; }
PYTHON
def square(n): return n * n
JAVA
static int square(int n) { return n * n; }

Arrays & Collections

C arrays are fixed-size and manually managed. Java arrays are fixed-size but bounds-checked. Python lists resize dynamically.

C
int nums[5] = {1, 2, 3, 4, 5}; int first = nums[0];
PYTHON
nums = [1, 2, 3, 4, 5] nums.append(6) # grows dynamically
JAVA
int[] nums = {1, 2, 3, 4, 5}; int first = nums[0];

Classes & Structs

C has no classes, only plain structs with no methods. Python and Java are both properly object-oriented.

C
struct Point { int x; int y; };
PYTHON
class Point: def __init__(self, x, y): self.x = x self.y = y
JAVA
class Point { int x, y; Point(int x, int y) { this.x = x; this.y = y; } }

Memory Model

C gives you manual control (malloc/free) and no safety net. Java and Python both use automatic garbage collection.

C
int *p = malloc(sizeof(int)); *p = 42; free(p); // your responsibility
PYTHON
p = [42] # memory freed automatically # once nothing references it
JAVA
Integer p = 42; // memory freed by the // garbage collector automatically

Error Handling

C has no exceptions — it relies on return codes. Python and Java both use structured try/except (or try/catch).

C
FILE *f = fopen("data.txt", "r"); if (f == NULL) { printf("Could not open file\n"); }
PYTHON
try: f = open("data.txt") except FileNotFoundError: print("Could not open file")
JAVA
try { var f = new FileReader("data.txt"); } catch (IOException e) { System.out.println("Could not open file"); }
